# Empowered Women in STEM – Ambition Project

## Overview

This project is a web development project created to celebrate and empower women in Science, Technology, Engineering, and Mathematics (STEM).
The website highlights inspiration, motivational quotes, and images of women who represent strength and innovation in STEM fields.

The goal of this project is to combine creativity and technology while promoting the importance of diversity and empowerment in STEM.

---

## Technologies Used

* **HTML** – for the structure of the webpage
* **CSS** – for styling and visual design
* **JavaScript** – for interactive elements such as the inspirational quote generator

---

## Features

* Inspirational header about women in STEM
* A motivational quote section with a button that generates new quotes
* Images celebrating women in STEM fields
* Responsive layout with modern styling and animations

---

## Project Files

* `Ambition.html` – main webpage structure
* `Ambition.css` – styling and layout
* `Ambition.js` – JavaScript functionality

---

## Inspiration

This project was inspired by the values of **empowerment, leadership, and innovation** in STEM.
It celebrates women who are breaking barriers and shaping the future of science and technology.
